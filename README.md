# Customer-Churn-Prediction-Website

This project predicts whether a customer is likely to leave (churn) using Machine Learning.

## Features

- Logistic Regression ML model
- Predicts customer churn probability
- Web application built using Flask
- Simple and user-friendly interface
- Displays prediction result with probability

## Technologies Used

Python  
Pandas  
NumPy  
Scikit-learn  
Flask  
HTML  
CSS  

## How to Run Project

1. Clone repository

2. Install dependencies

pip install -r requirements.txt

3. Run application

python app.py

4. Open browser

http://127.0.0.1:5000

## Project Structure

app.py → backend logic  
index.html → user interface  
style.css → design  
churn_model.pkl → trained ML model  

## Problem Statement

Customer churn prediction helps companies identify customers who are likely to stop using their service. Businesses can take action to retain such customers.
